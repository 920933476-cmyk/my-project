# 口袋游泳教练 - 版本记录

---

## RC2 — 2026-06-18

### 新增
- **按钮移动到底部**：生成训练计划按钮移至表单最底部，填写完无需回滚顶部
- **Session 细化**：每条训练指令输出 7 个维度（配速/间歇/心率/Drill/目的/RPE/时长）
- **心率区间**：基于年龄自动计算 6 区心率系统，每条训练带目标心率范围
- **间歇时间**：组间休息、出发间隔、预计完成时间，精确到秒
- **Drill Registry V1**：58 个专业 Drill 知识库，按泳姿/水平/目标/伤病智能推荐

### 优化
- **训练计划专业度提升**：从"能生成计划"升级为"像专业教练写的计划"
- **结果区交互优化**：
  - 生成后自动滚动到结果区域
  - 结果顶部增加「返回修改」和「重新生成」按钮
  - 移动端适配（按钮高度 ≥48px）
- **Rule Engine V1 集成**：知识库驱动训练计划生成，不同输入输出完全不同的计划

### 为什么？
RC1 的计划内容偏概括性描述，用户拿到后不知道具体怎么游。RC2 让每一条指令都可直接执行——配速多少、间歇多久、心率控制在什么范围、练什么动作、为什么这么练。

### 以后？
- **V2**：Drill 详情页（点击展开动作分解 + 视频链接）
- **V3**：用户反馈闭环（"这节课太难了/太简单了" → 自动调整下次计划难度）
- 你可能做到 **RC10**

## RC1 — 2026-06-17

### 新增
- **MVP 上线**：口袋游泳教练首个可发布版本
- **Rule Engine V0**：基于表单输入生成差异化训练计划
- **多维度输入**：年龄/性别/水平/项目/目标/时间/泳池长度/周量/天数/器材/伤病
- **4 大模块**：热身 → 技术 → 主集 → 放松
- **技术重点 + 陆上训练 + 恢复建议 + 计划解释**
- **纯文本复制**：一键复制训练计划为文本格式

### 优化
- **UI 基础版**：表单 + 结果双栏布局
- **时间格式兼容**：支持"1分15秒"/"1:15"/"75秒"等多种输入
- **表单验证**：必填项校验 + 错误提示

### 为什么？
从 Coze Agent 迁移到独立网页应用，让用户无需打开对话即可获取专业训练计划。
